# fixtures/

Сломанные пакеты для тестирования агента. Каждый пакет — отдельный сценарий с реальными симптомами.

---

## broken-vitest-config — `@kb-labs/agent-analytics`

**Симптом:** `pnpm test` падает при запуске. Тесты не находятся или не выполняются — зависит от того с какой директории запускаешь. Сам код в `src/analytics.ts` рабочий, тест написан корректно.

---

## broken-tsconfig — `@kb-labs/agent-scheduler`

**Симптом:** `pnpm build` завершается с ошибкой. tsup не может найти конфиг или падает на этапе компиляции. При этом `src/queue.ts` компилируется чисто — проблема не в коде.

---

## broken-imports — `@kb-labs/agent-memory`

**Симптом:** `pnpm build` падает с ошибками о ненайденных модулях. Несколько импортов в `src/store.ts` и `src/session-index.ts` не резолвятся. Часть из них — несуществующие пакеты, часть — неправильный путь внутри проекта.

---

## broken-build — `@kb-labs/agent-reporter`

**Симптом:** `pnpm build` проходит частично — один из entrypoints собирается, второй падает с ошибкой о ненайденном файле. В итоге `dist/` неполный.
